# Iris Flower Classification

## Project Description
This project uses Machine Learning to classify Iris flowers into:
- Setosa
- Versicolor
- Virginica

based on sepal and petal measurements.

## Technologies Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-Learn

## Algorithm Used
- Random Forest Classifier

## Features
- Data Analysis
- Data Visualization
- Model Training
- Accuracy Evaluation
- Species Prediction

## Accuracy
97% - 100%

## How to Run

1. Install dependencies

pip install pandas matplotlib seaborn scikit-learn

2. Run

python iris.py
